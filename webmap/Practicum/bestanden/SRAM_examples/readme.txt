This directory contains:

- samsung_sram.pdf
datasheet of the SRAM that is on the livedesign evaluation kit.


- sram.vhd
A simple and inaccurate model of the SRAM.
This can be used for simulation of the design.

- write_SRAM
Contains an example how you can write to both SRAMs (source file, constraint file and sof file)

- read_SRAM
Contains an example how you can read from both SRAMs (source file, constraint file and sof file).
